# Config Files

Repository for organizing my macOS configuration files.

## Structure

- **yabai/**: Configuration for the [yabai](https://github.com/koekeishiya/yabai) window manager

  - BSP (Binary Space Partitioning) layout
  - Window opacity
  - Padding and gaps
  - Mouse settings
- **skhd/**: Configuration for the [skhd](https://github.com/koekeishiya/skhd) keyboard shortcut daemon

  - Shortcut to start/stop yabai (Ctrl + Alt + Space)
- **.zshrc**: Configuration for the zsh with some alias, plugins and more
